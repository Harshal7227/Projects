# 🧠 Memory Card Game

A web-based memory-testing card game built using **HTML**, **CSS**, and **JavaScript**. This interactive game helps players train their memory by matching pairs of hidden image cards on a grid.

---

## 🎯 Objective

Flip cards two at a time to find matching image pairs. The goal is to match all the pairs using the fewest number of moves and in the shortest time.

---

## 🚀 Features

- 🧩 Dynamic 4x4 card grid (16 cards - 8 image pairs)
- 🔄 "New Game" button to reshuffle and restart the game
- 🔢 Move counter to track the number of attempts
- ⏱️ Timer to track game duration
- 🎉 "Congratulations" message when all pairs are matched
- ✅ Fully responsive layout for mobile and desktop
- 💡 Clean, commented code with scalable structure

---

## 🖥️ Technologies Used

- **HTML5** — Page structure and card layout
- **CSS3** — Grid styling, card animations, responsive design
- **JavaScript** — Game logic: card shuffling, matching, moves, and timer

---

## 🕹️ How to Play

1. Click on any card to flip it and reveal the hidden image.
2. Flip another card to try to find a match.
3. If the two cards match, they remain face-up.
4. If they don't, they flip back after a short delay.
5. Continue flipping cards to find all matching pairs.
6. Track your **Moves** and **Time** during the game.
7. Press **"New Game"** to reset and try again.
